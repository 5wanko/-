# 単語帳アプリ

SwiftUIで開発した、出題範囲をピンポイントで自由指定できる、モダンで汎用的な単語学習・クイズアプリです。

既存の単語帳アプリにありがちな「100語単位の固定出題」ではなく、「今日の通学中に1〜20番だけ完璧にしたい」といった個人個人の細かな学習ニーズにフィットするツールを目指して開発しました。

---

## ✨ 主な機能 (Features)

- **柔軟な出題範囲の指定**: 「開始番号」から「終了番号」まで、1問単位でテスト範囲を自由にカスタマイズ
- **CSVインポート機能**: お手持ちの単語リスト（CSV形式）を読み込んで、独自の単語帳を瞬時に作成
- **iOSネイティブ音声（TTS）**: `AVFoundation` を活用した、ネイティブ発音の自動読み上げ
- **選べる学習モード**: 直感的に解ける「4択クイズ」と、じっくり復習できる「単語帳」の2モード搭載
- **テストのカスタマイズ**: シャッフル出題のON/OFF、10秒の時間制限、効果音・音声再生の切り替え機能

---

## 🛠️ テックスタック (Tech Stack)

- **Language:** Swift 5
- **Framework:** SwiftUI
- **Audio/Speech:** AVFoundation (AVSpeechSynthesizer)
- **Data Architecture:** CSVデータパースによる動的な単語データ管理

---

## 🚀 使い方 (How to Use)

1. 本リポジトリをクローンまたはダウンロードし、Xcodeでプロジェクトを開きます。
2. お手持ちの単語データを `単語,意味` のシンプルなCSV形式で用意します。
3. アプリ内の「CSVファイルを追加読み込み」からインポートすることで、すぐにあなた専用のテスト環境が完成します。
   *(※本リポジトリのソースコード自体には、特定の書籍などの商用単語データは一切含まれていません。)*
